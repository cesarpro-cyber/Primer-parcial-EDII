# Proyecto: Árbol Binario de Productos

## 📌 Descripción
Este proyecto implementa un **árbol binario de búsqueda (ABB)** en Java para gestionar productos identificados por un `id` y un `nombre`.  
El sistema permite insertar, buscar y recorrer productos en orden, demostrando el uso de estructuras de datos y programación orientada a objetos.

---

## 🚀 Funcionalidades
- **Insertar productos** en el árbol binario según su `id`.
- **Buscar productos** por identificador.
- **Recorrido InOrden** para mostrar los productos ordenados.
- Ejemplo práctico con productos como *Laptop, Mouse, Teclado, Monitor, USB*.

---

## 🛠️ Tecnologías utilizadas
- **Lenguaje:** Java (JDK 8+)
- **Paradigma:** Programación Orientada a Objetos
- **Estructura de datos:** Árbol Binario de Búsqueda
